# Endpoints for /records

## Species
### `/records/{genus}/{species}`
`GET`: Return a list of records for a given species.

<RequestExample
    baseEndpoint={"/records"}
    defaultPaths={["Camponotus", "fragilis"]}
/>


## Subspecies
### `/records/{genus}/{species}/{subspecies}`
`GET`: Return a list of records for a given subspecies.
