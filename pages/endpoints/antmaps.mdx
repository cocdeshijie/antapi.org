# Endpoints for /antmaps

## Genus
### `/antmaps/{genus}`
`GET`: Returns an antmaps image for the given genus.

## Species
### `/antmaps/{genus}/{species}`
`GET`: Returns an antmaps image for the given species.

## Subspecies
### `/antmaps/{genus}/{species}/{subspecies}`
`GET`: Returns an antmaps image for the given subspecies.
